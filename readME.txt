
1. Install anaconda
2. Install Pytorch from Anaconda Prompt.
3. Make a separate environment for experimentation.
4. Activate that environment.
5. Install pyTorch here.
6. Make the environment available in the Jupyter Notebooks by installing ipykernal.
7. Install pandas.
8. Open Jupyter notebook.
9. Change kernel to the environment we just created and run modelrunner_tester.py.

Please modify the path as needed for the movie_reviews.tar and wiki-news-300d-1M.vec files in modelrunner_tester.py
before step 9.
